* The styling service is used to provide custom styles to components by dependency injection.

* Can be provided directly in the component's tag using the ngt-stylizable directive or from your aplication main module.

* When provided in the main module, styling will be adopted as a standard for all components related to it. You will need the style name to provide for the right component.
