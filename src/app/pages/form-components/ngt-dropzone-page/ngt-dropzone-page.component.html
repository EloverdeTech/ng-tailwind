Component used to preview, upload and download attachments.

Note: This component needs the service NgtAttachmentHttpService to be provided for it.


Examples:

NgModel value: {{ ngModelOne | json }}
NgModel value: {{ ngModelTwo | json }}
{{ codeExample }}

Inputs:

multipleSelection

Determines whether you can select more than one file at a time.

itemsLimit

Number of attachments allowed.

showFileName

Show or hide file name.

disableClick

Disable click and allows only drag and drop.

disabled

Enable or disable component.

removable

Determines if the attachments can be removed.

verticalExpandable

Enable or disable vertical expand.

acceptedFiles

Determines which files are accepted. Must be an array of mime types.

maxFileSize

Maximum size allowed for each attachment.

previewType

Determines the component preview type. Must be instance of NgtDropzonePreviewType.

isRequired

Determines if the component data is required.

label

Component label.

name

Name used by NgModel. Must be unique.

remoteResource

The resource used to make remote requests.


Outputs:

onFileSelected

Event emitted when a file is selected.

onFileSelectError

Event raised when an error occurred while selecting a file.

onFileUploaded

Event emitted when file upload has finished.

onFileUploadFail

Event raised when an error occurred while uploading a file.

onFileRemoved

Event emitted when a file is removed.

onFilePreviewLoaded

Event emitted when file preview loaded has finished.


Public attributes:

loading

Enable or disable loading mode.

shining

Enable or disable shining mode.


Enums:

NgtDropzonePreviewType

The preview type provided for the component must be one of this.

DEFAULT

Uses the standard viewing method.

IMAGE

Uses the visualization method optimized for images.

VIDEO

Uses the visualization method optimized for videos.


NgtDropzoneErrorType

The errors reported in the output onFileSelectError and onFileUploadFail are instances of this enum.

DEFAULT

Type used when the error is unknown.

SIZE

Type used when the limit size has been exceeded.

NO_MULTIPLE

Type used when the selection is not multiple and more than one file is selected at a time.

ITEMS_LIMIT

Type used when the allowed items limit is exceeded.

TYPE

Used when the file mime type is not allowed.


Methods:

openFileSelector

Method to open NgtDropzone file selector.
