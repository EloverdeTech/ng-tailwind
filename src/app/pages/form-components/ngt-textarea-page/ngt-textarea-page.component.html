Component used for text entry in forms.


Examples:

NgModel value: {{ ngtTextarea }}
{{ codeExample }}

Visual Inputs:

label

Textarea label.

placeholder

Textarea placeholder.

shining

When true, the component will display a shining mode.

rows

Defines the number of predefined lines to be displayed in the component.

helpTitle

Component help title.

helpText

Determines if help text is enable.


Behavior Inputs:

isDisabled

When true, the component will be disabled.

isReadonly

When true, the component will be readonly.

name

Name used by NgModel. Must be unique.

focus

Focuses textarea automatically after view init.

jit

When true, the ng model will be updated at typing time.


Validation Inputs:

isRequired

Determines whether the field is required.

maxlength

Defines the maximum number of characters accepted.


Methods:

setFocus

Focuses the NgtTextarea.

clear

Clears the textarea value.
