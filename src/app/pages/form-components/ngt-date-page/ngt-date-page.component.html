Component used to select a specific date and hour or a date range.


Examples:

NgModel value: {{ dateOne }}
NgModel value: {{ dateTwo }}
NgModel value: {{ dateRange }}
{{ codeExample }}

Visual Inputs:

label

Component label.

placeholder

Component placeholder.

helpTitle

Component help title.

helpText

Determines if help text is enable.

shining

When true, the component will display a loading mode.

dateFormat

Date format to show for the user

dateFormatNgModel

Date format to use in NgModel


Behavior Inputs:

name

Name used by NgModel. Must be unique.

isDisabled

When true, the component will be disabled.

isReadonly

When true, the component will be readonly.

mode

Determines the selection mode. Must be a NgtDateMode instance.

time_24hr

When true, the time will has 24 hours format.

enableTime

When true, the time selection will enable.

minuteIncrement

Determines the minute increment. Default is 1.

allowInput

When true, the user can set a value manually.

locale

Determines the date location. Must be a NgtDateLocale instance.


Validation Inputs:

isRequired

Determines whether the field is required.


Enums:

NgtDateLocale

The locale provided for the component must be one of this.

BRAZIL

The component will use the portuguese as default language.

US

The component will use the english as default language.


NgtDateMode

The mode provided for the component must be one of this.

SINGLE

The component will be used to select only one date.

RANGE

The component will be used to select a range of dates.
