Component used for select an option in forms.


Examples:

{{ item ? item.fontendText : '' }} NgModel value: {{ ngtSelect | json }}
{{ item ? item.fontendText : '' }} NgModel value: {{ ngtSelectMultiple | json }}
{{ codeExample }}

Visual Inputs:

label

Component label.

placeholder

Component placeholder.

helpTitle

Component help title.

helpText

Determines if help text is enable.

shining

When true, the component will display a shining mode.

loading

When true, the component will display a loading mode.

loadingText

Text displayed during loading.

notFoundText

Text displayed when the result not founded.

dropdownPosition

Set the dropdown position on open. Can be: auto, bottom and top. Default is auto.

typeToSearchText

Placeholder text for search.

clearAllTooltip

Tooltip text for clear selected options.

createText

Set custom text when allow creating.

labelForId

Id to associate control with label.


Behavior Inputs:

isDisabled

When true, the component will be disabled.

name

Name used by NgModel. Must be unique.

remoteResource

The resource used to make remote requests.

hideSelected

When true, hide the selected elements.

allowCreate

Allows to create custom options. Default false.

bindLabel

When the component is used in a way that its items are static, that is, passed in @Input() items, bindLabel defines the object property that should be used in the provided object to obtain the text that should be displayed to the user for each item in the array.

bindValue

Object property to use for selected model. By default binds to whole object.

items

Array de itens utilizado para renderizar opções estáticas.

closeOnSelect

Whether to close the menu when a value is selected. Default true.

clearable

Allow to clear selected value. Default true.

groupBy

Allow to group items by key or function expression.

groupValue

Function expression to provide group value.

maxSelectedItems

When multiple is true, allows to set a limit number of selection.

multiple

Allows to select multiple items.

searchable

Allow to search for value. Default true.

clearSearchOnAdd

Clears search input when item is selected. Default true. Default false when closeOnSelect is false.

virtualScroll

Enable virtual scroll for better performance when rendering a lot of data. Default true.

trackBy

Provide custom trackBy function.

tabIndex

Set tabindex on NgtSelect.

typeahead

Custom autocomplete or advanced filter.


Validation Inputs:

isRequired

Determines whether the field is required.


Methods:

setFocus

Focuses the NgtSelect.

refresh

Reload the options in NgtSelect.

reset

Reset the NgtSelect to its initial state.

getFilterInputValue

Get the current filter value of the NgtSelect.
