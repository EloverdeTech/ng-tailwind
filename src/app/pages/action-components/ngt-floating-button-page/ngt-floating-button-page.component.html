Component used to display a series of options when clicking on a floating button. Can be used for navigation and for execute anything method when clicking an a option.

{{ codeExample }}
{{ optionsCodeExample }}

Inputs:

menus

Array of menus.

icon

Floating button icon.

label

Floating button label.

withAnimation

Determines whether the button has animation.


Public Attributes:

isOpen

When true the menu is opened, otherwise it is closed.

ngtStyle

Provides styling via NgtStylizableService.


Enums:

NgtFloatingButtonType

The type provided for the component must be one of this.

NAVIGATION

The menu will be used for navigation and need a "url" parameter.

ACTION

The menu will be used for perform a function and need a "action" parameter.


Interfaces:

NgtFloatingButtonMenu

The menus provided for the component must follow this interface.

name(?)

Menu name.

tooltip(?)

Menu tooltip.

icon(?)

Menu icon.

type

Determines whether the menu will be used for navigation or action when clicked.

action(?)

Function that must be performed when clicking on the menu (must be type ACTION).

externalLink(?)

When true, the link provided will open in a new tab (must be type NAVIGATION).

url(?)

URL to be open when click (must be type NAVIGATION).
