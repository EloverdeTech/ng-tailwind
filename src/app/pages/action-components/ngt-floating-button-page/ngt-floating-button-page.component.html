Component used to display a series of options when clicking on a floating button. Currently is only usable for navigation.


Examples:

{{ codeExample }}

Inputs:

menus

Array of menus.

icon

Floating button icon.

label

Floating button label.

withAnimation

Determines whether the button has animation.


Public Attributes:

isOpen

When true the menu is opened, otherwise it is closed.

ngtStyle

Provides styling via NgtStylizableService.


Interfaces:

NgtFloatingButtonMenu

The menus provided for the component must follow this interface.

externalLink

When true, the link provided will open in a new tab.

url

URL to be open when click.

icon

Menu icon.

tooltip

Menu tooltip.

name

Menu name.
