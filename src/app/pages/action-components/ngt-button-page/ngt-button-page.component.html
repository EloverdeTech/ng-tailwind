Component used to perform an action when clicking on an button. Can be used too for navigation.


Examples:

Success Danger Warning Information
{{ codeExample }}

Inputs:

href

Address that will be used in the router link when the link option is enable.

link

When enabled it makes the button behave like an anchor tag.

type

Change the background color according to the chosen type (success, danger, warning, info).

loading

Controls the button loading status.

isDisable

Determines whether the component is enable.

noSubmit

When true, prevents the form from being triggered when the button is clicked.
